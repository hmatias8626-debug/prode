# Prode Odds con botoncitos

App Streamlit para consultar cuotas desde The Odds API con accesos rápidos:

- Mundial 2026
- Campeón Mundial
- Libertadores
- Sudamericana
- Chile Primera
- Brasil Serie B
- España Segunda
- Japón J League
- Liga Argentina manual por CSV/Excel

## Secrets

En Streamlit Cloud:

```toml
ODDS_API_KEY = "TU_API_KEY"
```
