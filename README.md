# Prode Mundial 2026

App Streamlit para cargar cruces del Mundial 2026, consultar cuotas desde The Odds API y calcular probabilidades reales para un prode.

## Archivos necesarios

- app.py
- requirements.txt
- README.md

## Secrets

En Streamlit Cloud:

Settings → Secrets

Pegar:

```toml
ODDS_API_KEY = "TU_API_KEY"
```

No subas tu API key real a GitHub.
