# Prode Odds - Multi Categoría

App Streamlit para consultar cuotas desde The Odds API por categoría:
- Mundial 2026
- Copa Libertadores
- Copa Sudamericana
- ligas disponibles en la API
- mercados de campeón/outrights cuando existan

## Archivos

- app.py
- requirements.txt
- README.md

## Secrets

En Streamlit Cloud:

```toml
ODDS_API_KEY = "TU_API_KEY"
```

## Main file path

```text
app.py
```
